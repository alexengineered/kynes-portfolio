# kynes-portfolio
A refined, editorial-style portfolio featuring elegant Cormorant Garamond serif typography against a soft beige background (#EFF0E8) with muted earth tone accents. Clean single-column layout emphasizes content hierarchy through carefully scaled type and generous spacing. Minimal color palette (grays, soft greens) creates a calm, sophisticated atmosphere. Includes smooth transitions, subtle hover states, and responsive design. Built with semantic HTML and CSS custom properties. Perfect for engineers seeking an understated, literary presentation that prioritizes readability and timeless aesthetic over interactive effects.
